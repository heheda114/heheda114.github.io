网站是如何搭建：

+1、服务器(硬件)

+2、容器(环境)(服务)和他的标配

	+a、IIS(IIS容器是每台Windows系统自带的容器)  
			+ASP  
			+Asp.net  
			+ASHX(和Asp.net为两兄弟)  
			+ASA  
			+CER  
			+CDX  
	+b、Apache  
			+php
	+c、Tomcat  
			+JSP(java语言所写)
	+d、Nginx  
			+php
	+e、..........

+3、脚本和他主要搭配的数据库

	+a、ASP
	    +Accss  
	    +SQL server (MSSQL)  
	 +b、ASP.NET(ASPX)  
	     +SQL server (MSSQL)(适用于大中型企业)
	  +c、PHP  
	     +MYSQL(免费、开发简便、适用于中小型企业)  
	   +d、JSP  
	      +Oracle(收费、开发需求大、性能牛批、适用于大型企业类似于政府、银行)

+4、数据库

+a、Access
+b、SQL server (MSSQL)
+c、MYSQL
+d、Oracle

+标配关系表

+IIS+ASP+Access(70%)(主要用于企业网站后台数据存储)  
	+IIS+ASP+SQL server(数据库量一般比较大，用于大型数据存放)  
+Apache+php+Mysql(90%)
+Nginx+PHP+Mysql(90%)  
+Tomcat+JSP+Oracle(20%)  
	+Tomcat+JSP+Mysql(80%)
